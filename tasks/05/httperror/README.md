# HTTP ошибка

Вам необходимо реализовать тип ошибки `httpError` и функции, помогающие работать с ним.

Требования:
1. Ошибка должна нести информацию об http коде ответа.
2. Функция `GetStatusCode` должна возвращать код ответа, содержащийся в ошибке. Если ошибка, переданная в функцию, не является `httpError`, необходимо вернуть `http.StatusInternalServerError`.
3. Запрещено менять сигнатуры функций и методов (но разрешено добавлять свои).
4. Ошибка должна корректно работать с цепочками ошибок, в частности, оборачивать какую-либо другую через `WrapWithHTTPError`.
